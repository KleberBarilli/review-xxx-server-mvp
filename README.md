# 🎮 ReviewX - Game & Media Tracker API

> **Note:** This repository represents the **MVP (Minimum Viable Product)** and the initial version of this project. The development has since continued and evolved into a newer, updated version that currently powers the live platform.

🌐 **Check out the live project:** [loggvault.com](https://loggvault.com)

A RESTful API built with **Java 21** and **Spring Boot**, designed to manage, track, and analyze various forms of media, including video games, TV series, books, and movies. This project focuses on automatically importing and centralizing data directly from Steam, utilizing efficient caching and database persistence.

---

## 🚀 Technologies Used

- **Java 21**
- **Spring Boot 3.x**
- **Spring Data MongoDB**
